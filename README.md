# 📊 Sales & Marketing Analysis Dashboard | Power BI

## 📌 Project Overview

This project focuses on analyzing sales and marketing performance through an interactive Power BI dashboard. The objective was to transform raw transactional data into meaningful business insights by implementing ETL processes, data modeling, and KPI reporting.

The solution was developed using Power BI, Power Query, and DAX, following a Star Schema approach consisting of one centralized fact table and multiple dimension tables.

---

## 🎯 Problem Statement

Businesses often struggle to monitor sales performance, customer behavior, marketing effectiveness, and product trends because data is stored across multiple sources and lacks a proper analytical structure.

Without a centralized reporting system, decision-makers face challenges in:

- Tracking revenue and sales performance.
- Identifying top-performing products.
- Measuring marketing campaign effectiveness.
- Understanding customer purchasing behavior.
- Analyzing sales across different channels and regions.
- Generating actionable insights for business growth.

---

## 💡 Solution

To address these challenges, an end-to-end Power BI solution was developed.

The project involved:

- Performing data cleaning and transformation using Power Query.
- Building a Star Schema data model.
- Creating one Fact Table and multiple Dimension Tables.
- Generating surrogate keys using Index Columns.
- Establishing one-to-many relationships between dimension and fact tables.
- Creating DAX measures for KPI calculations.
- Designing interactive dashboards with filters, slicers, and drill-through capabilities.
